# 🎯 AI Office
## Multi-Agent Software Development Framework

<Notes>
Welcome to AI Office! A complete framework for coordinating multiple AI agents through software development lifecycle.
</Notes>

---

## What is AI Office?

- 📋 **File-based virtual agency system**
- 👥 **27 specialized agents** with personalities & skills
- 📊 **Kanban task board** with milestones & tracking
- 🛡️ **Loop guards** to prevent infinite cycles
- 📝 **Artifact management** (PRD, ADR, runbooks)
- ⚖️ **Italian Legal Studio agency** (NEW in v1.4.0)

---

## The Pipeline

```
router → prd → adr → plan → tasks → dev → qa → review → release → postmortem
                                ↘ ux_research → design_ui ─────────────────┘
                                               ↘ security → dev/qa
```

<Notes>
Standard software studio pipeline. Can be customized per agency (lean-startup, game-studio, etc.)
</Notes>

## Loop Guards

Prevent infinite QA/review/UAT cycles:

| Loop | Trigger | Max | Escalation |
|------|---------|-----|------------|
| **qa → dev** | Regression | 2 | Planner |
| **review → dev** | Revision | 2 | Planner |
| **uat → dev** | UAT fail | 1 | Planner |

<Notes>
Hard limits. When reached, task → BLOCKED with explicit unblock criteria. Prevents endless cycles.
</Notes>

## Quick Start

### 1️⃣ Install
```bash
./install.sh [project-path]
./setup.sh [project-path]
```

### 2️⃣ Start a Feature
```bash
/office:route Add real-time notifications
```

### 3️⃣ Create Milestone
```bash
/office:milestone create M1 "Notifications" tasks:yes
```

### 4️⃣ Execute Pipeline
```bash
/office:scaffold notifications prd
/office:advance notifications prd "PRD approved"
# ... continue through pipeline ...
```

Simple workflow. Route → scaffold → advance through stages. Agents handle each stage.
</Notes>
